【概要】
　パラメータから戦闘結果をシミュレートするソフトウェアデース！
　先人たちが残した種々のデータを元にしています。感謝しますね！
　ちょっと(記述量が)重い感じもしますけれど、でも、頑張ります！
　お役に立ちましたか？　そう、なら、頑張った甲斐がありました！

【もうちょっと詳しい概要】
・既知の検証結果から、艦これの戦闘をシミュレートすることができます。
・平均残り耐久、勝率などの統計的なデータを収集することも可能です。
・このソフトウェアは同名CUI版のフロントエンドです。
　装備や艦娘や艦隊データの編集、そしてシミュレート実行までを
　GUI上で実行することが出来ます。
・ランタイムエラーが出た際は、
　「Visual Studio 2015 の Visual C++ 再頒布可能パッケージ」
　(https://www.microsoft.com/ja-jp/download/details.aspx?id=49984)
　をインストールすれば動くはずです。

【操作説明】
○装備データタブ
　装備データを編集できます。左欄に装備一覧が表示され、クリックするとその項目が
読み込まれて右欄に表示されます。以下、ボタンの意味について説明します。
　・追加：右欄で入力したデータを末尾に追加します。
　・削除：左欄で選択した装備を削除します。
　・元に戻す：操作をやり直す……のではなく、右欄のデータを左欄の選択した装備の
　　　　　　　もので上書きします。修正をミスった場合にどうぞ。
　・再読み込み：weapon.csvを再読み込みします。右欄を初期化したい時にもどうぞ。
　・上書き保存：編集結果をweapon.csvに反映します。これを押さないと修正・追加・
　　　　　　　　削除した結果が失われ、シミュレートデータとして利用できません。
○艦娘データタブ
　艦娘データを編集できます。左欄に艦娘一覧が表示され、クリックするとその項目が
読み込まれて右欄に表示されます。以下、ボタンの意味について説明します。
　・追加：右欄で入力したデータを末尾に追加します。
　・削除：左欄で選択した艦娘を削除します。
　・元に戻す：右欄のデータを左欄の選択した艦娘のもので上書きします。
　・再読み込み：kammusu.csvを再読み込みします。右欄を初期化したい時にもどうぞ。
　・上書き保存：編集結果をkammusu.csvに反映します。これを押さないと修正・追加・
　　　　　　　　削除した結果が失われ、シミュレートデータとして利用できません。
○艦隊データタブ
　艦隊データを編集できます。左欄に現在の艦隊に所属する艦娘一覧が表示され、
クリックするとその項目が読み込まれて右欄に表示されます。また、制空値・索敵値は
変更の度に自動計算されます(索敵値計算アルゴリズムは最新版を使用)。なお、艦隊名と
司令部レベルは左欄の操作では変わらない、艦隊共通のものです。
　以下、ボタンの意味について説明します。
　・元に戻す……右欄のデータを左欄の選択した艦娘のもので上書きします。
　　　　　　　　この際、艦載機数・残り耐久・cond値・残り弾薬量も初期化されます。
　・↑と↓……左欄で選択した艦娘の順番を変更します。なお、一番上が旗艦です。
　・追加・更新……編集結果を反映します。前2タブと異なり、ファイルを保存しないと
　　　　　　　　　編集した結果も無意味になりますのでご注意ください。
　・リセット……左欄を消去します。当然右欄も初期化されます。
　・削除……左欄で選択した艦娘を削除します。
○戦闘シミュレーションタブ
　戦闘をシミュレーションします。第1艦隊(先攻)・第2艦隊(後攻)を読み込み、
シミュレーション設定をした後に「計算開始」ボタンをクリックすれば、
シミュレーションした結果が右下のテキストボックスに表示されます。
　ここで、「先後入れ替え」チェックボックスにチェックを入れておくと、陣形情報も
込みで先後逆になりますのでご注意ください。
　なお、シミュレーション結果の詳細については、CUI版のReadmeを参照のこと。
○海域シミュレーションタブ
　海域出撃をシミュレーションします。出撃する艦隊・出撃する海域を読み込み、
シミュレーション設定をした後に「計算開始」ボタンをクリックすれば、
シミュレーションした結果が右下のテキストボックスに表示されます。
　なお、シミュレーション結果の詳細については、CUI版のReadmeを参照のこと。

○メニューボタン
　・開く……艦隊データのファイルを開く。
　・名前を付けて保存、上書き保存……艦隊データのファイルを保存する。
　・終了……ソフトウェアを終了する。
　・バージョン情報……バージョン情報を表示する。
